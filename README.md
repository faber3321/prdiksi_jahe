# Prediksi Harga Jahe Berbasis Web

Aplikasi web ini menggunakan Recurrent Neural Network (RNN) untuk memprediksi harga jahe dengan integrasi API cuaca dan dashboard analitik interaktif.

## Fitur Utama
- Prediksi harga jahe menggunakan model RNN
- Integrasi data cuaca melalui API
- Dashboard analitik interaktif
- Visualisasi data harga dan cuaca

## Teknologi yang Digunakan
- Backend: Python (Flask/Django)
- Machine Learning: TensorFlow/Keras untuk RNN
- Frontend: HTML, CSS, JavaScript
- Visualisasi: Chart.js atau D3.js
- API Cuaca: OpenWeatherMap atau API lainnya
